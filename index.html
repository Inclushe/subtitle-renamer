Subtitle Renamer

More Info

Subtitles will be copied then renamed to the same name as its corresponding episode and saved to a .zip file.

Example: Given an episode titled "[GrubsYes] Spy x Family - 01 (1080p) [C00FFEE].mkv" and a subtitle file titled "SPYÃ—FAMILY.S01E01.SHRimp.Jetflix.ja[cc].srt", the subtitle file will be renamed "[GrubsYes] Spy x Family - 01 (1080p) [C00FFEE].srt".

Episodes and subtitles will be mapped by episode number or, if we detect an issue, by file order.

All files are handled locally, and no files are sent over the network.
